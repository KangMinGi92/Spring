프로그램만들어보기!
학생관리프로그램 -java-
학생정보를 관리하는 프로그램
기능 : 학생등록,학생수정,학생삭제,학생전체조회,학생 이름으로 조회
학생정보 : 이름,학번,학년,전공,주소,성별

mvc패턴으로 구조를 설정
m : model -> 데이터를 관리하는 기능/역할을 클래스들
v : view -> 사용자에게 보여주는 화면을 출력하는 기능/역할을하는 클래스들
c : controller -> 클라이언트(프로그램이용자) 요청한 서비스를 진행하는 기능/역할을 하는 클래스들

클래스를 설계할때는 단일책임의 원칙이 있음.
화면을 보여주는 클래스1개
서비스를 진행하는 클래스 1개
클라이언트가 저장한 정보를 관리하는 클래스1개
프로그램을 실행시키는 클래스1개
데이터를 관리하는 클래스1개
